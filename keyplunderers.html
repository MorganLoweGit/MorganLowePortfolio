Key Plunderers

Summary

  • Brief: First year university project.
  • Timeline: September 2023 - January 2024.
  • "An introduction to Unity."
My role:

Key Plunderers is my first 2D Unity Project. This project was my first experience with Unity, C#, and revision control using GitHub.

  • After learning basic C#, I began with working on the player movement scripts and key objects. This involved using RididBody and BoxCollider components, and learning how they can be utilised.
  • Designed sprites for the character animations in Paint 3D, and imported them into Unity.
  • After creating the animation handlers I created a RigidBody box that can be pushed along the floor, that can activate a secret trigger on the left side of the map to unlock a speed multiplier power up.
  • Created a goal object that progresses the player once they unlock it with the corresponding colour-coded key.
  • Implemented co-op gameplay with a controller, and the ability to change from keyboard to controller on the main character.
