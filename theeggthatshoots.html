The Egg that Shoots

Summary

  • Brief: Second year university project.
  • Timeline: September 2024 - January 2025.
  • "First experience with C++"
My role:

This game was my first project working with C++ and the SDL2 library.

  • Like my python game, I built a custom animation system using a png spritesheet, where I also drew up my own pixel art assets. I used a txt file to store data each object in the spritesheet.
  • The tilemap is formed of individual images from a spritesheet formed together into a single surface.
  • My project was structured so each class had it's own header and cpp file, improving optimisation and readability.
  • Used a delta time system to make the game FPS independent.
  • Created a UI healthbar that depletes when the player takes damage.
  • Used a technique called object pooling with my enemy and bullet objects so the game ran at a higher framerate and used less memory.
