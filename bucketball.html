Bucket Ball

Summary

  • Brief: First year university project.
  • Timeline: January 2024 - June 2024.
  • "Understanding the workflow of 3D game design."

Left click: rotate (hold to pause rotation), scroll wheel: zoom

My role:

Bucket Ball is a game I developed using Unreal Engine 5, using the in-built blueprint system. I learnt many workflows during my time in development, such as animations, asset building and particle systems.

  • White-box prototyping: Planning out the setting that the player will roam.
  • Used Unreal Engine 5's landscape tools combined with a custom built landscape material that automatically changes texture based on landscape steepness.
  • Through development I am also practicing revision control with GitHub.
  • Created my own blend spaces to handle animations imported from Mixamo.
  • UI system to display the score and current level to the player.
  • My workflow for asset creation was modelling and UV mapping using Maya, then painting textures with Substance Painter 3D.
  • I used Photoshop to create the target textures in the goal props.
  • Finally, I created particles using Unreal Engine's Niagara Particle System that trigger when the player scores.
